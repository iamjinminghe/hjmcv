## ✨ Features

- **🌐 Bilingual Support**: Seamless switching between English and Simplified Chinese without page reloads.
- **📱 Fully Responsive**: Optimized for desktop, tablet, and mobile viewing using CSS Grid and Flexbox.
- **✨ Smooth Animations**: Integrated scroll-reveal animations using the vanilla `IntersectionObserver` API.
- **🚀 Zero Dependencies**: Built entirely with Vanilla HTML5, CSS3, and JavaScript. No build tools or heavy frameworks required.
- **🎨 Modern UI/UX**: Clean typography, card-based layouts, and integrated SVG icons for a professional look.

## 🛠️ Tech Stack

- **Markup**: HTML5
- **Styling**: CSS3 (Custom Properties/Variables, Flexbox, Grid)
- **Scripting**: Vanilla JavaScript (ES6+)

## 🚀 Quick Start & Deployment

Since this portfolio is built with static files, it is incredibly easy to deploy using **GitHub Pages**.

1. **Fork this repository** (or clone it to your local machine).
2. **Customize the content**: Open `index.html` and replace the text, links, and icons with your own information.
3. **Deploy via GitHub Pages**:
   - Go to your repository **Settings**.
   - Navigate to the **Pages** section on the left sidebar.
   - Under **Build and deployment**, select the `main` branch and the `/root` folder.
   - Click **Save**. Your site will be live in a few minutes!
